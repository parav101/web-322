The ways to send data to server & access the data inside the routes on server:

  1. using query string in url, e.g. /users?occupation=student&page=8&perPage=10
    - to access the data on server: req.query.occupation, req.query.page, req.query.perPage; OR all query parametes in one JavaScript object: req.query
  2. using http request body, sent from e.g. Add new user form: name=John and age=22, ...
    - to access the data on server: req.body.name, req.body.age; OR all form data in one JavaScript object: req.body
  3. using parameter in route path, e.g. route: /users/12 (id=12), /users/25
    - to access the data on server: req.params.id; OR all parameters in one JavaScript object: req.params
  4. using cookies

Web app (pages) vs Web API (json data)

  • res.send(): /users?occupation=student&page=8&perPage=10
  • res.json(): /users?occupation=student&page=8&perPage=10&json4api=yes
